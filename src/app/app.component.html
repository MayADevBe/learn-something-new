Learn Something New

Want to learn something new? But don't know what? This small project is for exactly this situation. It will give you a random suggestion - maybe even something that will suprise/interest you!

How to Use?
1. Just click on the button 'I want to learn something new!'
2. When selecting the categories, you can select the main(first) category to select all subcategories. (You can also select only the main category, it might contain things not fitting in the subcategories.)
3. Then select the time you want to spend. (These are just very rough estimations.)
4. Then you select how much money you want to spend. (Again, very rough estimations.)
Generally, the categories can be seen as very subjective and it can vary greatly how much time/money you would actually spend. Since in the end you choose how much time/money you want to put into it. There are also some things to learn, where it might depend on if you want to learn only the theory or also apply it.
